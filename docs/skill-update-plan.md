# /update-plan スキル

議事録・Slackログなどの最新の議論内容を受け取り、企画骨子ファイルを自動更新して変更サマリーを出力するカスタムコマンド。

## 使い方

Claude Codeを起動してプロンプトに入力：

```
/update-plan

【企画骨子】docs/2026-05-CSカルテDB企画.md

【議論内容】
（議事録やSlackログをそのままペースト）

【追加資料】
（任意）
```

## 何をしてくれるか

| ステップ | 内容 |
|----------|------|
| 1. 現状把握 | 企画骨子・議論内容・追加資料を読み込む |
| 2. 差分分析 | 決定事項・変更点・持ち越しを特定 |
| 3. 骨子更新 | 変更を反映して保存・更新履歴を追記 |
| 4. サマリー出力 | 変更内容をチャットに出力 |
| 5. push | git commit & push まで自動実行 |

## 出力フォーマット（変更サマリー）

```
## 変更サマリー

### ✅ 反映した変更
- 〇〇を△△に修正
- □□を追加

### ⚠️ 要確認・持ち越し
- ××については次回MTGで確認

### 📄 更新ファイル
- docs/2026-05-CSカルテDB企画.md
```

## コマンドファイルの場所

`.claude/commands/update-plan.md`
